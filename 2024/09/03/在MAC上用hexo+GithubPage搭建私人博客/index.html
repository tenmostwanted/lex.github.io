在MAC上用hexo+GithubPage搭建私人博客

踩坑点记录

  • 没有权限的问题。
    sudo -i / sudo -s 切换一下。

  • npm install 下载失败
    换一下npm config里面的源，换成国内源即可。

  • Github 权限
    虽然是，SSH 可以连上，但是hexo d部署的时候，还是显示权限问题；
    换了另一种，PAT，加个Token，然后还是https的repo，输入密码的时候把密码改为token。
